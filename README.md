这是一个通过调用官方接口，能够提供网易云音乐每日听满300首歌曲的基于PHP语言的API项目。

强烈建议配合python全自动脚本项目使用更佳：https://github.com/ZainCheung/netease-cloud

## 灵感来自

[Binaryify/NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

## 功能特性

1. 登录
2. 签到
3. 查询用户信息
4. 听完300首歌曲
5. 刷单曲播放次数
